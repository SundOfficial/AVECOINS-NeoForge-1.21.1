# AVECOINS 💰

[![NeoForge Version](https://img.shields.io/badge/NeoForge-1.21.1-blue)](https://neoforged.net/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**AVECOINS** adds a set of **material‑based coins** to use as a currency or decorative item in your Minecraft world or server.  

From common copper to netherite, each coin has its own color, tooltip, and crafting chain.

---

## ✨ Features

- **5 coin tiers**: Copper, Iron, Gold, Diamond, Netherite.
- **Consistent crafting recipes** – upgrade 4 coins + 1 ingot/gem to get the next tier.
- **Informative tooltips**:
    - Normal tooltip describes the coin.
    - Hold `SHIFT` to see how to upgrade it.
- **Custom creative tab** `AVECOINS`.
- **Fully localised** – English (`en_us`), Spanish (`es_es`), Mexican Spanish (`es_mx`).
- **Lightweight** – no dependencies, works on servers and clients.

---

## 📦 Coins & Crafting

| Coin          | Upgrade recipe                                         | Reverse recipe                  |
|---------------|--------------------------------------------------------|---------------------------------|
| **Copper**    | Base coin (no upgrade from lower)                      | 1x Iron Coin -> 4x Copper Coins |
| **Iron**      | 4x Copper Coins + 1x Iron Ingot → 1x Iron Coin                 | 1x Iron Coin → 4x Copper Coins  |
| **Gold**      | 4x Iron Coins + 1x Gold Ingot → 1x Gold Coin              | 1x Gold Coin → 4x Iron Coins    |
| **Diamond**   | 4x Gold Coins + 1x Diamond → 1x Diamond Coin              | 1x Diamond Coin → 4x Gold Coins |
| **Netherite** | 4x Diamond Coins + 1x Netherite Ingot → 1x Netherite Coin | 1x Netherite Coin → 4x Diamond Coins |

> All upgrade recipes use the **shaped** pattern shown below.  
> 
> Reverse recipes are **shapeless** (just put the high‑tier coin in a crafting grid).

## 📥 Installation

1. Install **NeoForge** for Minecraft 1.21.1.
2. Download `avecoins-neoforge-1.21.1-<version>.jar` from the [Releases](https://github.com/yourusername/avecoins/releases) page.
3. Place the `.jar` into your `mods` folder.
4. Launch the game.
